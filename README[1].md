# FIFA World Cup 2026 Prediction Challenge - V14 Concept

## Status
Landing page approved. V14 updates the next pages.

## V14 changes
- Removed the separate Bracket tab/page.
- Integrated the tournament path/bracket into the Predictions page.
- Replaced Group A/B/C blocks with one chronological fixture list.
- Each fixture is one horizontal bar with group, date, CET time, venue, flags, team names and score inputs.
- Live Results lists all games chronologically.
- Live Table is the player leaderboard, not football tournament standings.

## Notes
This is a conceptual static GitHub Pages build. It stores sample predictions locally in the browser. Firebase/email/live API can be connected later.


## V15 Concept Update

- Landing page remains approved.
- Prediction page keeps the chronological group-stage fixture bars.
- Separate bracket-page concept remains removed.
- Tournament path is now integrated into the prediction page using the same horizontal-bar format as group fixtures.
- Round of 32, Round of 16, quarter-finals, semi-finals and final each have a round heading followed by one horizontal match bar per fixture.
- User picks winners inside those horizontal bars, and selected winners continue to flow through the later rounds.
