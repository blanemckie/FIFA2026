# FIFA World Cup 2026 Prediction Challenge - V62

V62 adds a “Just Browsing” guest mode on the landing page so visitors can explore the prediction game without registering first. Guest users can browse the Prediction Centre and other pages, but must register before saving or locking predictions.

Also retains the V61 score-input auto-jump behaviour, V60 condensed mobile scoring guide, production Firebase collections, validation, and mobile layout improvements.
