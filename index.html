🏗️ From Zero to Working Prototype

No previous website-building experience • ~2 hours of build time • Voice dictation + ChatGPT • GitHub Pages + Firebase • £0 cost

Inspired by Damien's AI lunch chats, I decided to see how quickly I could turn an idea into a working product.

Before starting this project (and even now!), I had no prior experience building websites, hosting applications, using GitHub, configuring Firebase or developing front-end code. Everything you see here was learned and built through curiosity, experimentation and AI-assisted development.

This page tracks the main steps, approximate time taken and example prompts used to show how simple it was to get from idea to prototype.

🏆 Top 3 Tips

🎤 1. Dictation makes it easy

Get started by speaking in practical language. Explain what you want as if you were talking to a colleague.

🧠 2. Prompt practice makes perfect

Use memory and remind AI of previous decisions, especially if errors or mistakes keep repeating, so you do not have to keep explaining the same things over and over again.

🧱 3. Build in batches

Create batches before making changes. It saves time, keeps iterations cleaner and makes the project easier to steer.
