🤖 AI Probability Predictor

🚧
Work in Progress
This page is still being built and tested — numbers, layout and features may be incomplete or change without notice.

How this works: Each team's strength rating is built from how it actually performed in the group stage (points won, goal difference). The real Round of 32 draw is then resolved from the final group standings, and the bracket is simulated forward hundreds of times — each simulated match decided by the two teams' relative strength, not a coin flip. For any round that's already been played, the real result is used instead of a simulation. In every single simulation, each player's locked-in points are added to whatever they'd score from that simulated outcome, and whoever has the highest total in that simulation is the "winner" of that run. The percentage shown is simply how often a player comes out on top across all the simulations — so a real points lead carries its full weight, it isn't diluted just because there are still a lot of points left on the table.

Recalculates automatically every time the Live Table refreshes with new results — so these percentages move daily as the tournament unfolds.

🚀 How I Built This

Inspired by Damien's AI lunch chats, I built this World Cup prediction game mainly through dictation to ChatGPT over a handful of evenings.

The challenge was simple: could someone with limited AI experience and no web design or hosting experience create a fun prediction game for friends, family and colleagues as quickly as possible using AI?

The first thing I asked ChatGPT to do was track the time spent on each stage and create a “How I Built This” page so the journey could be shared transparently and show just how accessible this technology has become.

💡 Biggest Learning

Whilst I was surprised at how quickly something like this could be turned into a working product, the biggest learning was how effective voice dictation became.

Being able to describe changes naturally rather than typing detailed specifications sped up the development process exponentially.
