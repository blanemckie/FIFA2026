🛠️ From Idea to Kick-Off 🛠️

Inspired by Damien's AI lunch chats, I built this World Cup prediction game using mainly dictation to ChatGPT over a couple of evening sessions.

One of my first prompts was to ask ChatGPT to track the approximate time spent and number of prompts used, so I could document the build journey on this page.

The goal wasn't to build a perfect solution. It was to see how quickly an idea could become something real using AI, despite limited experience with website development, GitHub, Firebase and front-end design.

🏆 Top 3 Tips

🎤 1. Dictation makes it easy

Get started by speaking in practical language. Explain what you want as if you were talking to a colleague.

🧠 2. Prompt practice makes perfect

Use memory and remind AI of previous decisions, especially if errors or mistakes keep repeating, so you do not have to keep explaining the same things over and over again.

🧱 3. Build in batches

Create batches before making changes. It saves time, keeps iterations cleaner and makes the project easier to steer.
